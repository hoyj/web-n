CSS

Cascading Style Sheets (CSS) is a style sheet language used for describing the presentation of a document written in a markup language. Although most often used to set the visual style of web pages and user interfaces written in HTML and XHTML, the language can be applied to any XML document, including plain XML, SVG and XUL, and is applicable to rendering in speech, or on other media. Along with HTML and JavaScript, CSS is a cornerstone technology used by most websites to create visually engaging webpages, user interfaces for web applications, and user interfaces for many mobile applications.

  • class 선택자

    .(dot)을 사용하며 사용 방식은 .[class명] 이다.
    예를 들어 <h1 class="saw">가 있다면
    .saw를 통해 saw라는 class를 가지고 있는 태그들만 선택을 할 수 있다.

    복수의 선택자를 사용할 수 있는데, style 코드블럭 내A,B가 있다고 할때, 각 상황에 따라 다르게 적용 된다.

    • A B 둘다 사용하되 겹치는 값이 없는 경우 : A B 둘다 적용
    • A B 둘다 사용하되 겹치는 값이 있는 경우 + A가 B보다 먼저 정의된 경우: B의 값이 적용
    • A B 둘다 사용하되 겹치는 값이 있는 경우 + B가 A보다 먼저 정의된 경우 : A의 값이 적용

  • id 선택자

    #(hash tag)를 사용하며 사용 방식은 #[id명]이다.
    예를 들어 <h1 id="saw">가 있다면
    #saw를 통해 saw라는 id를 가지고 있는 태그들만 선택을 할 수 있다.

    그렇다면 id와 class의 차이는 무엇일까요? id 는 한번 밖에 사용하도록 권장한다는 것입니다. 우선 순위는
    태그 < class < id
    입니다.

Box model

화면 전체를 사용하는 박스를 block level element(element는 태그의 동의어)라 하며 자기 자신의 컨텐트의 크기만큼을 가지고 있는 것을 inline element라고 한다.

각 태그의 box 종료는 display 속성을 통해 바꿀 수 있으며, display 에 가능한 값은

  • inline
  • block
  • none
가 존재하며, none을 사용할 경우 해당 태그에 감싸인 컨텐트 자체가 화면에서 보이지 않는다(자리 또한 차지 하지 않는다)

box 크기를 커스터마이징하고 싶다면 width, height 속성을 통해 바꿀 수 있다.

[안쪽]컨텐트와 테두리(border)사이의 간격은 padding 키워드를 이용한다.
[바깥쪽]테두리와 테두리 사이의 간격은 margin 키워드를 이용한다.

box model 용어: margin > border > padding > content

더 궁금한 것은 css box model을 검색!
브라우저에 내장되어 있는 element inspector 또한 적극적으로 활용하자!
